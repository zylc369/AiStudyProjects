# Lists Test

This file tests various list formats.

## Unordered Lists

### Simple Unordered List

- Item 1
- Item 2
- Item 3

### Unordered List with Different Bullet Styles

* First item
* Second item
* Third item

+ Alternative bullet 1
+ Alternative bullet 2
+ Alternative bullet 3

## Ordered Lists

### Simple Ordered List

1. First item
2. Second item
3. Third item

### Ordered List with Numbers

10. Tenth item
11. Eleventh item
12. Twelfth item

## Nested Lists

### Unordered Nested List

- Level 1 item 1
  - Level 2 item 1.1
  - Level 2 item 1.2
    - Level 3 item 1.2.1
    - Level 3 item 1.2.2
  - Level 2 item 1.3
- Level 1 item 2
  - Level 2 item 2.1
- Level 1 item 3

### Ordered Nested List

1. First level item
   1. Second level item 1
   2. Second level item 2
2. First level item 2
   1. Second level item 1
   2. Second level item 2
      1. Third level item 1
      2. Third level item 2
3. First level item 3

### Mixed Nested List

1. Ordered level 1
   - Unordered level 2
   - Another unordered level 2
     1. Ordered level 3
     2. Another ordered level 3
2. Back to ordered level 1
   - Unordered level 2
     - Deeper unordered level 3
3. Final ordered level 1
